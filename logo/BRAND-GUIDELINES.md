# HumberStack Brand Guidelines - Master v1.0

## 🎨 Visual Identity - Master Version 1.0

### Brand Name
**HumberStack** (capital H, capital S)
- "Humber" - Foundation, reliability, infrastructure
- "Stack" - Technology stack, layered solutions, modern architecture
- Orange accent on "Stack" emphasizes the technical focus

### Logo Concept
The HumberStack logo uses the simple geometric pattern `|||=` - three vertical bars beside three horizontal bars. This minimal design represents:

- **Vertical bars (|||)**: Structure, foundation, building blocks
- **Horizontal bars (=)**: Layers, stack, infrastructure
- **Combined**: Parallel processing, efficiency, modern tech architecture
- **Minimal aesthetic**: Clean, geometric, instantly recognizable

### Symbol Meaning
- **Six elements**: Simplicity through modularity
- **Perpendicular lines**: Different perspectives, comprehensive solutions
- **Equal spacing**: Balance, precision, engineering excellence
- **Geometric purity**: Modern, timeless design

## ✍️ Typography

### Font Stack
```css
font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, 
             'Helvetica Neue', Arial, sans-serif;
```

### Type Scale
- **Heading 1**: 48px / 700 weight
- **Heading 2**: 36px / 700 weight
- **Heading 3**: 24px / 600 weight
- **Body**: 16px / 400 weight
- **Small**: 14px / 400 weight

### Guidelines
- Use system fonts for performance
- Maintain hierarchy with size and weight
- Keep line length between 50-75 characters
- Line height: 1.5-1.6 for body text

## 🖼️ Imagery Style

### Photography
- Clean, modern, professional
- Bright and optimistic
- Technology-focused but human
- Avoid overly corporate stock photos

### Icons
- Simple, outlined style
- Consistent stroke width
- Align to 24px grid
- Use brand colors sparingly

## 📱 Digital Presence

### Website Style
- Clean, minimal design
- Generous white space
- Clear visual hierarchy
- Mobile-first responsive

### Social Media
- Professional headshots or logo
- Consistent color scheme
- High-quality images
- Educational and helpful content

## 💬 Tone & Voice

### Do's
✅ Be helpful and educational
✅ Explain technical concepts simply
✅ Show personality
✅ Be confident in expertise
✅ Use active voice

### Don'ts
❌ Use excessive jargon
❌ Be condescending
❌ Overpromise
❌ Use tired buzzwords
❌ Be overly salesy

### Example Phrases
- "Let's build something great together"
- "Fast, secure, and reliable"
- "Technology that works for you"
- "Simplifying complex solutions"

## 📐 Layout Principles

### Spacing
- Use consistent increments (8px base)
- Generous padding around elements
- Clear separation between sections
- Mobile: 16px side margins minimum
- Desktop: Maximum 1200px content width

### Grid
- 12-column grid system
- Flexible, responsive breakpoints
- Align elements to grid
- Maintain visual rhythm
