## Design Specifications - Master v1.0

### Icon Design: |||=
- **Concept**: Three vertical bars (|||) beside three horizontal bars (=)
- **Symbolism**: 
  - Vertical bars: Building blocks, foundation, structure
  - Horizontal bars: Layers, stack, infrastructure
  - Together: Parallel processing, efficiency, modern architecture
- **Style**: Minimal, geometric, clean, compact
- **Dimensions**:
  - Vertical bars: 16-18px width × 60-80px height
  - Horizontal bars: 30-35px width × 16-18px height
  - Spacing: 6px between vertical bars, 6-7px between horizontal bars
- **Colors**: 
  - Background: #1a1a1a (dark)
  - Bars: #ffffff (white)
  - Corner radius: 3-4px for smooth edges

### Typography
- **Brand Name**: HumberStack (capital H, capital S)
- **Font**: Helvetica Neue (600 weight)
- **Size**: 64px (main logo), 48px (horizontal logo)
- **Letter spacing**: -2.5px
- **Color**: 
  - "Humber": #1a1a1a (dark) / #ffffff (white version)
  - "Stack": #f97316 (orange accent)
- **Kerning**: Custom tspan adjustments for optical perfection

### Color System
- **Primary Dark**: #1a1a1a - Icon background, text
- **Primary Light**: #ffffff - Icon elements, white logo
- **Accent Orange**: #f97316 - "Stack" text highlight
- **Transparent**: Used for overlay effects

### Canvas Sizes
- **Main logo with text**: 550×140px
- **Horizontal logo**: 650×200px
- **Square icon**: 200×200px
- **Favicon**: 32×32px

## File Naming Convention

Format: `humberstack_{type}_v{version}_{variant}_{status}.svg`

- **type**: logo, icon, favicon, wordmark
- **version**: 1.0, 1.1, 2.0, etc.
- **variant**: helvetica_neue_text, white, black, color
- **status**: master, draft, deprecated

## Modification Procedures

### To Create a New Version:
1. **DO NOT modify master files directly**
2. Copy master file to new version number
3. Make changes to new version
4. Test thoroughly
5. Update this VERSION-CONTROL.md
6. Lock new version as master
7. Mark old version as deprecated if applicable

### To Make Variations:
- Create variant files (e.g., _blue, _inverted, _simplified)
- Reference base master version in comments
- Do not lock variants as masters

## Usage Guidelines

### Primary Use Cases
- **humberstack_v1.0_helvetica_neue_text_master.svg**: Website headers, marketing materials, presentations
- **humberstack_v1.0_white_master.svg**: Dark backgrounds, video overlays, dark mode UI
- **humberstack_icon_v1.0_master.svg**: Social media profiles, app icons, avatars
- **humberstack_favicon_v1.0_master.svg**: Browser favicons, bookmarks

### Export Recommendations
- **Web**: Use SVG directly (preferred)
- **Favicon**: Export to ICO and PNG (16x16, 32x32, 48x48)
- **Social Media**: Export to PNG (400x400, 1200x630)
- **Print**: Export to high-res PNG or use SVG
