## Maintenance Notes

### When to Use v1.0 vs v2.0

**Use v1.0** (Font-based):
- When editing text content
- Making design changes
- Creating new variations
- Source file management

**Use v2.0** (Paths):
- Production websites
- Final deliverables
- Distribution to clients
- Anywhere font compatibility matters

### Future Updates
If logo text needs changes:
1. Edit v1.0 font-based files
2. Run `convert-text-to-paths.ps1`
3. Run `optimize-v2.ps1`
4. Run `validate-v2-quality.ps1`
5. Update version numbers if needed
