# Cloudflare Websites - Humberstack Project

Complete setup for humberstack.com including website and email configuration.

## 📁 Project Structure

```
cloudflare-websites/
├── humberstack/        # Website files (HTML, CSS, JS)
├── email-setup/        # Email routing configuration
└── logo/               # Brand assets and logo files
```

## 🚀 Quick Start

### 1. Deploy Website
Navigate to the [humberstack](./humberstack) directory:
- Complete, responsive website ready for Cloudflare Pages
- See [humberstack/README.md](./humberstack/README.md) for deployment instructions

### 2. Set Up Email
Navigate to the [email-setup](./email-setup) directory:
- Free Cloudflare Email Routing configuration
- See [email-setup/QUICK-START.md](./email-setup/QUICK-START.md) for 5-minute setup

## 📋 What's Included

### Website (humberstack/)
- Modern, responsive design
- Home, About, and Contact pages
- Contact form ready for email integration
- Cloudflare Pages optimized
- Security headers pre-configured

### Email Setup (email-setup/)
- Cloudflare Email Routing guides
- Email Worker for sending emails
- DNS configuration reference
- Wrangler deployment config

### Logo & Branding (logo/)
- SVG logo files (multiple variations)
- Favicon and app icons
- Brand guidelines and color palette
- PNG generation script

## 🎯 Next Steps

1. **Review the logo files** in the logo directory
